# BrainAcademy Static Campaign Page

Statyczna strona kampanii przygotowana pod GitHub Pages.

## Struktura

```text
.
|- index.html
|- 404.html
|- assets/
|  |- css/
|  |  |- styles.css
|  |- js/
|  |  |- main.js
|  |- img/
|     |- icon-512.png
|     |- island-unlocked.svg
|     |- rocket.svg
|     |- screen-codie.png
|     |- screen-puzzle.png
|     |- screen-streak.png
|     |- tlo.png
|     |- unlock.svg
```

## Publikacja na GitHub Pages

1. Wypchnij repozytorium na GitHub.
2. Wejdz w `Settings -> Pages`.
3. W `Build and deployment` wybierz:
   - `Source: Deploy from a branch`
   - Branch: `main` (lub `master`)
   - Folder: `/ (root)`
4. Zapisz ustawienia i poczekaj na publikacje.
